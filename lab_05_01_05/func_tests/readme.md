# Тесты для лабораторной работы №05_01_05

## Входные данные
stdin - целые числа до инвалидного ввода

## Выходные данные
max_count - максимальная длина последовательности из однаковых чисел

## Позитивные тесты:
- 01 - 1 1 1 1 1 2 3 (первая последовательность максимальная)
- 02 - 1 2 2 2 2 2 3 (последовательность посередине максимальная)
- 03 - 1 2 3 3 3 3 3 (последовательность в конце максимальная)
- 04 - 1 2 3 4 5 6 7 (все поседовательности - один)
- 05 - 1 1 2 2 3 3 4 4 (все последовательности - два)
- 06 - 1 (одно число)
- 07 - -1 -1 -1 -2 -3 (первая максимальная с отрицательными)
- 08 - -1 -2 -2 -2 -3 (посередине максимальная с отрицательными)
- 09 - -1 -2 -3 -3 -3 (последняя максимальная с отрицательными)
- 10 - -1 2 -2 2 -2 3 (неверная максимальная последователность из двоек)
- 11 - -1 2 2 -3 -3 -3 (положительные и отрицательные числа)
- 12 - -1 (одно отрицательное число)

## Негативные тесты:
- 01 - (пустой ввод)
- 02 - a (сразу инвалидный ввод)
- 03 - a 1 2 2 3 (сначала инвалидный, потом валидный)
- 04 - a b c (несколько инвалидных вводов)

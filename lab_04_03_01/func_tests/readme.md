# Тесты для лабораторной работы №04_03_01

## Входные данные
Строка str - входная строка

## Выходные данные
Строка new_str - выходная строка

## Позитивные тесты:
- 01 - hello world hi jack (разделенные пробелом)
- 02 - hello,world,hi,jack (разделенные запятой)
- 03 - hello;world;hi;jack (разделенные точка с запятой)
- 04 - hello:world:hi:jack (разделенные двоеточием)
- 05 - hello-world-hi-jack (разделенные тире)
- 06 - hello.world.hi.jack (разделенные точкой)
- 07 - hello!world!hi!jack (разделенные восклицательным знаком)
- 08 - hello?world?hi?jack (разделенные вопросительным знаком)
- 09 - hello   world   hi    jack (несколько пробелов)
- 10 - hello ,.;world    hi:?! jack (знаки чередуются)
- 11 - hello like table like chair like (несколько слов как последних)
- 12 - aaaaaa bbbbbb cccccc (слова из одного символа)
- 13 - hello liiiiike liiiike like (несколько слов, после удаления символов будут равны последнему)

## Негативные тесты:
- 01 - hello (одно слово)
- 02 - hello hello hello (одинаковые слова)
- 03 -          (только пробелы)
- 04 - ,,,,,,,, (только запятые)
- 05 - ;;;;;;;; (только точки с запятой)
- 06 - :::::::: (только двоеточия)
- 07 - -------- (только тире)
- 08 - ........ (только точки)
- 09 - !!!!!!!! (только восклицательные знаки)
- 10 - ???????? (только вопросительные знаки)
- 11 - (пустая строка)
- 12 - (переполнение)

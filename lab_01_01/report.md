## Пункт 1. Ошибка в объявлении header - файла
Был упущен знак **#** в объявлении **#include** и знаки "<" и  ">", а также неправильное название самого файла - **stdio.h**, а не **studio.h**
```
include studio.h  -->  #include <stdio.h>
```

## Пункт 2. Объявление main
Не указан тип возвращаемого объекта для **main** и перепутаны местами круглые и фигурные скобки. Не указано что принимает функция **main** на вход.
```
main{}          int main(void)
(               {
...      -->    ...
)               }
```

## Пункт 3. Присваивание значения для s
В присваивании есть лишний символ **":"**. Достаточно **s = 0;**.
```
s: = 56;  -->  s = 56;
```

## Пункт 4. Неправильное обращение в printf
Команда **print** вместо **printf**. Нет кавычек, а также нет спецификатора преобразования **%d** для целого **s**.
```
print (Year has s weeks)  -->  printf("Year has %d weeks\n", s);
```

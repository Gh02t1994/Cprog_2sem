# Тесты для лабораторной работы №01_06_05

## Входные данные
Вещественные x_p, y_p, x_q, y_q, x_r, y_r, x_s, y_s
x_p - координата x для точки p
y_p - координата y для точки p
x_q - координата x для точки q
y_q - координата y для точки q
x_r - координата x для точки r
y_r - координата y для точки r
x_s - координата x для точки s
y_s - координата y для точки s

## Выходные данные
Целое 1 или 0

## Позитивные тесты:
- 01 - 1 2 3 2 2 1 2 3 (два пересекающихся отрезка)
- 02 - 1 2 3 2 2 2 2 3 (конец одного отрезка лежит на другом отрезке)
- 03 - 1 2 3 2 1 1 3 1 (два непересекающихся отрезка)
- 04 - 1.1 2.2 3.3 2.2 2.2 1.1 2.2 3.3 (вещественные значения)
- 05 - 1 4 4 4 1 5 4 5 (два горизонтальных параллельных)
- 06 - 5 1 5 4 6 1 6 4 (два вертикальных параллельных)
- 07 - 0 0 4 0 2 -1 2 1 (вертикальный пересекает горизонтальный)


## Негативные тесты:
- 01 - aa (вместо чисел буквы)
- 02 - 1 2 3 2 2 3 2 3 (вместо отрезка точка)

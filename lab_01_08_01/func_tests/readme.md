# Тесты для лабораторной работы №01_08_01

## Входные данные
Беззнаковые целые byte_1, byte_2, byte_3, byte_4

## Выходные данные
Беззнаковое 32-битное целое в двоичном виде total_byte; Беззнаковые целые res_b_1, res_b_2, res_b_3, res_b_4

## Позитивные тесты:
- 01 - 0 <= byte_1 и byte_2 и byte_3 и byte_4 <= 255

## Негативные тесты:
- 01 - вместо чисел буквы
- 02 - byte_1 или byte_2 или byte_3 или byte_4 < 0
- 03 - byte_1 или byte_2 или byte_3 или byte_4 > 255

# Тесты для лабораторной работы №01_08_01

## Входные данные
Беззнаковые целые byte_1, byte_2, byte_3, byte_4
byte_1 - первый байт
byte_2 - второй байт
byte_3 - третий байт
byte_4 - четвертый байт

## Выходные данные
Беззнаковое 32-битное целое в двоичном виде total_byte; Беззнаковые целые res_b_1, res_b_2, res_b_3, res_b_4
total_byte - результат упаковки 4 байт в 2-ом виде
res_b_1 - распакованный обратно первый байт
res_b_2 - распакованный обратно второй байт
res_b_3 - распакованный обратно третий байт
res_b_4 - распакованный обратно четвертый байт

## Позитивные тесты:
- 01 - 0 0 0 1 (минимальное ненулевое значение)
- 02 - 0 0 0 0 (все нули)
- 03 - 255 255 255 255 (максимальное значение)
- 04 - 1 0 0 0 (только старший байт)
- 05 - 0 1 0 0 (только второй байт)
- 06 - 0 0 1 0 (только третий байт)
- 07 - 1 2 3 4 (последовательные числа)
- 08 - 128 64 32 16 (степени двойки)
- 09 - 255 0 255 0 (чередование макс/мин)
- 10 - 1 1 1 1 (все единицы)
- 11 - 255 0 0 0 (только старший байт максимальный)

## Негативные тесты:
- 01 - aa (вместо чисел буквы)
- 02 - 256 0 0 0 (первый байт выходит за диапазон)
- 03 - 0 256 0 0 (второй байт выходит за диапазон)
- 04 - 0 0 256 0 (третий байт выходит за диапазон)
- 05 - 0 0 0 256 (четвертый байт выходит за диапазон)
- 06 - -1 0 0 0 (отрицательное значение)
- 07 - 0 -1 0 0 (отрицательное значение)
- 08 - 0 0 -1 0 (отрицательное значение)
- 09 - 0 0 0 -1 (отрицательное значение)

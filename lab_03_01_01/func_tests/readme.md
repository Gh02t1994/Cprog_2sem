# Тесты для лабораторной работы №03_01_01

## Входные данные
Целый row_size - количество строк матрицы
Целый col_size - количество столбцов матрицы
Целые matrix[row][col] - элементы матрицы

## Выходные данные
Целые result_arr[row] - результаты симметричности строк

## Позитивные тесты:
- 01 - 3 1  1  2  3 (один столбец)
- 02 - 1 1  1 (один элемент)
- 03 - 3 2  1 1  2 2  3 3 (все строки симметричные)
- 04 - 3 3  1 2 1  1 2 3  3 4 3 (нечетное количество столбцов)
- 05 - 3 4  1 2 2 1  1 2 3 1  3 4 4 3 (четное количество столбцов)
- 06 - 3 3  1 2 3  4 5 6  7 8 9 (ни одна строка не симметрична)
- 07 - 3 3  -1 -2 -1  -1 2 1  1 -2 1 (с отрицательными числами)
- 08 - 1 6  1 2 3 3 2 1 (одна строка)
- 09 - 2 3  0 1 0  0 0 0 (с нулями)

## Негативные тесты:
- 01 - a 1 (первый размер - буква)
- 02 - 1 a (второй размер - буква) 
- 03 - -1 1 (первый размер отрицательный)
- 04 - 1 -1 (второй размер отрицательный)
- 05 - 11 1 (первый размер вне предела)
- 06 - 1 11 (второй предел вне предела)
- 07 - 0 1 (первый размер ноль)
- 08 - 1 0 (второй размер ноль)
- 09 - -1 -1 (оба размера отрицательные)
- 10 - 11 11 (оба размера вне пределв)
- 11 - 0 0 (оба размера нули)
- 12 - 1 2  1 a (элемент - буква)

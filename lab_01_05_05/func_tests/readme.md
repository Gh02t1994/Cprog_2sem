# Тесты для лабораторной работы №01_05_05

## Входные данные
Целое number - натуральное число

## Выходные данные
Выведенное число number - натуральное число

## Позитивные тесты:
- 01 - 5 (однозначное)
- 02 - 10 (с нулем на конце)
- 03 - 123 (трехзначное)
- 04 - 1000 (с несколькими нулями на конце)
- 05 - 100001 (несколько нулей посередине)
- 06 - 9999 (все цифры одинаковые)
- 07 - 2147483647 (максимальный int)

## Негативные тесты:
- 01 - 0 (нулевое)
- 02 - -5 (отрицательное)
- 03 - -123 (отрицательное многозначное)
- 04 - a (буква)

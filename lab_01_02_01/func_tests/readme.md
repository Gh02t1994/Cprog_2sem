# Тесты для лабораторной работы №01_02_01

## Входные данные
Вещественные side_a, side_b, angle_phi
side_a - первое основание трапеции
side_b - второе основание трапеции
angle_phi - угол в градусах при большем основании равнобедренной трапеции

## Выходные данные
Вещественное square
square - площадь трапеции, найденная по формуле формуле |a^2 - b^2| * tan(alpha) / 4

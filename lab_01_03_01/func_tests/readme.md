# Тесты для лабораторной работы №01_03_01

## Входные данные
Вещественные v_1, v_2, t_1, t_2
v_1 - число литров в первой емкости
v_2 - число литров в второй емкости
t_1 - температура воды в первой емкости
t_2 - температура воды во второй емкости

## Выходные данные
Вещественные v_total, t_total
v_total - сложенный объем воды из двух емкостей
t_total - получившаяся температура воды полученной смеси
